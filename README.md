# Enterprise Customer Intelligence & Predictive Analytics Pipeline 📊🔮

An end-to-end, production-grade data science pipeline designed to ingest messy corporate data, perform advanced data sanitization, and deploy dual machine learning models (Regression & Classification).

## 🚀 Key Features
* **Data Sanitization & Imputation:** Cleans complex string formats (currencies) and handles missing data using median imputation.
* **Feature Engineering:** Automated business logic integration for high-value client tagging.
* **Predictive Forecasting:** Multi-variable Linear Regression model to predict next-quarter revenue yields.
* **Account Classification:** Logistic Regression classifier to flag high-value VIP accounts with high probability confidence.

## 🛠️ Tech Stack Used
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib
* **IDE:** PyCharm

## 📊 Project Overview
This project simulates a real-world enterprise data environment. It converts raw, messy client data into actionable business intelligence, demonstrating proficiency in:
1. Handling `NaN` values and data cleaning.
2. Building and evaluating ML models.
3. Deploying production-ready predictive logic.
